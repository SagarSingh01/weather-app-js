# 🌦️ Weather App

A simple and interactive **Weather Application** built using **HTML, CSS, and JavaScript**.  
Search any city and get real-time weather details in a clean and responsive interface.

---

## 🚀 Features

- 🌍 Search weather by city name  
- 🌡️ Real-time temperature display  
- 💨 Shows humidity and wind speed  
- 📱 Fully responsive design (desktop + mobile)  
- ⚡ Fast and smooth API integration  
- 🎨 Clean and modern UI  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3 (Flexbox & Grid + Glassmorphism UI)  
- JavaScript (DOM manipulation & API handling)  
- OpenWeather API
