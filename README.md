🌦️ Weather App  
A simple and modern Weather Application built using HTML, CSS, and JavaScript.  
It fetches real-time weather data and displays temperature and location details in a clean, responsive UI.

---

🚀 Features  
🌍 Search weather by city name  
🌡️ Real-time temperature display  
💨 Shows humidity and wind speed  
📱 Fully responsive design (mobile + desktop)  
⚡ Fast and smooth API integration  
🎨 Clean and modern UI design  

---

🛠️ Tech Stack  
HTML5  
CSS3 (Flexbox & Grid + Glassmorphism UI)  
JavaScript (DOM manipulation & API handling)
